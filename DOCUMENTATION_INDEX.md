## ‚ùì FAQ

### Q: Where do I start?
A: Read `QUICKSTART.md` (2 minutes) then open the website

### Q: How do I log in?
A: Use demo account: admin / admin123 (or register)

### Q: What can admins do?
A: Create/edit/delete products and manage orders

### Q: What can regular users do?
A: Browse products, add to cart, checkout

### Q: Is the database persistent?
A: Yes! Uses MongoDB

### Q: Is it secure?
A: Yes! Passwords hashed, JWT tokens, role-based access

### Q: Can I run it locally?
A: Yes! Just run `npm start`

### Q: What if I find a bug?
A: Check TESTING.md troubleshooting section

### Q: Can I add more features?
A: Yes! Follow ARCHITECTURE.md to understand system

### Q: How is user authentication handled?
A: JWT tokens (7-day expiration) + bcrypt password hashing
